## 使用方法

### 直接打开
1. 双击 `Login_Page_Prototype.html` 文件
2. 在浏览器中打开即可查看原型

### 在线预览
将文件上传到服务器或使用本地服务器：
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx http-server
```

然后访问：`http://localhost:8000/Login_Page_Prototype.html`

## 与真实实现的对应关系

| 原型功能 | 真实实现 |
|---------|---------|
| 表单提交 | `handleSubmit` 函数 → `AuthContext.login()` |
| API调用 | 模拟延迟 → `POST /api/auth/login` |
| 错误提示 | `showError()` → Ant Design `Alert` 组件 |
| 加载状态 | 按钮状态切换 → `loading` state |
| 密码切换 | `togglePassword()` → Ant Design `Input.Password` |
| 背景动画 | CSS动画 → VANTA.js NET 效果 |
| 页面跳转 | `alert()` → `navigate('/')` |

## 注意事项

1. **原型演示**：这是一个静态原型，登录功能是模拟的，不会真正调用后端API
2. **浏览器兼容**：建议使用现代浏览器（Chrome、Firefox、Edge、Safari）
3. **移动端**：原型支持响应式设计，可在移动设备上查看
4. **性能**：网络背景动画可能在某些低性能设备上影响性能

## 后续开发建议

1. **真实API集成**：将模拟的登录逻辑替换为真实的API调用
2. **路由集成**：集成 React Router 实现真实的页面跳转
3. **状态管理**：使用 Context API 或 Redux 管理认证状态
4. **错误处理**：完善错误处理机制，包括网络错误、超时等
5. **安全性**：添加CSRF保护、XSS防护等安全措施
6. **可访问性**：添加ARIA标签，提升无障碍访问体验
